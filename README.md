# kafka-demo

Шпаргалка по работе с Apache Kafka. Примеры кода на PHP.

## Требования

* ```docker```
* ```docker-compose```
* ```make```
* ```bash```

## Управление

* ```make up``` - запуск Zookeeper & Kafka.
* ```make down``` - остановка Zookeeper & Kafka.
* ```make restart``` - перезапуск Zookeeper & Kafka.
